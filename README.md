### ・ML/CV
**Python/PyTorchによる機械学習・深層学習モデル開発経験**があります。<br>
環境構築にはAnacondaの仮想環境を使用し、画像処理やコンピュータビジョンタスクにおいてはOpenCVを活用した実装を行ってきました。

### ・Mobile App Dev
FlutterフレームワークとDart言語を用いた**クロスプラットフォームアプリのチーム開発経験**があります。<br>
プロジェクトでは主にiOSプラットフォームの対応、フロントエンド処理、およびUI/UXの実装を担当しています。<br>
また、Firebaseを用いた認証やデータベース連携などのバックエンド処理も自律的に構築できるレベルにあります。

### ・OS & Infra
メインの開発環境としてmacOSを使用しつつ、大規模なAI開発にはNVIDIA GPUを搭載したUbuntu（Linux）デスクトップ環境へSSH接続してリモートで作業を行っています。**大学研究でスパコンを使用**しており、Ubuntu/Linux環境における開発やリモートデスクトップ操作に関する知識、**データセンターやクラウド環境を用いた本格的なAI開発において即戦力となる実務能力**を備えています。<br>
全てのコードはGitで厳密にバージョン管理し、Dockerを用いて移植性の高い開発環境を構築できます。

### ・Tools & Docs
メインのエディタとしてVS Codeを活用しており、開発を効率的に行っています。<br>
また、技術的な知見を正確に共有するため、MarkdownやNotionを用いたドキュメント作成能力に長けています。特に大学の研究論文においてはLaTeXを用いて執筆しており、**専門的な文書作成やレポートの整形スキル**も備えています。
